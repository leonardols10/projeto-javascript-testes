## ✅ ETAPA 1: Estrutura do Projeto
**O que fotografar/printar:**
1. Abrir o VS Code
2. Ir em File → Open Folder
3. Selecionar: `C:\Users\Leonardo\Documents\Trabalho\trabalho full stack\projeto-javascript-testes`
4. Tirar print da estrutura exibida no Explorer (pasta src, tests, package.json, etc)

## ✅ ETAPA 2: Arquivos da Calculadora
**O que fotografar/printar:**
1. No VS Code, abrir o arquivo `src/calculadora.js`
2. Tirar print mostrando o código completo (ou em partes)

## ✅ ETAPA 4: Instalação de Dependências
**O que fotografar/printar:**
1. Abrir o terminal (Ctrl + `)
2. Executar: `npm install`
3. Tirar print mostrando a mensagem de sucesso com "added X packages"

## ✅ ETAPA 5: Execução dos Testes
**O que fotografar/printar:**
1. No terminal, executar: `npm test`
2. Tirar print mostrando:

## ✅ ETAPA 6: Testes com Cobertura
**O que fotografar/printar:**

## 🎯 DICAS PARA TIRAR PRINTS MELHORES:

1. **Use Shift + Win + S** para capturar apenas parte da tela
2. **Use Print Screen** para capturar a tela inteira
3. **Use Snags & Sketch** (Windows 11) para anotações nos prints
4. **Redimensione as janelas** para que fiquem legíveis
5. **Aumente o zoom** se necessário (Ctrl + +)
6. **Tire múltiplos prints** se o conteúdo não couber em um

---

## 💾 APÓS TIRAR TODOS OS PRINTS:

1. Organize os prints em pastas por ordem
2. Abra o **PowerPoint** ou **Word**
3. Crie um documento com todos os prints
4. Adicione descrições e títulos para cada etapa
5. Exporte como **PDF**

Pronto! Você terá um PDF profissional com toda documentação do projeto! 📄✨
