# Projeto de Calculadora com Testes Jest

## Descrição
Projeto de uma calculadora em JavaScript com testes unitários usando Jest.

## Funcionalidades
- ✅ Somar
- ✅ Subtrair
- ✅ Multiplicar
- ✅ Dividir
- ✅ Potência
- ✅ Raiz Quadrada

## Testes
Todos os testes estão implementados com Jest, cobrindo:
- Casos de sucesso (happy path)
- Casos extremos
- Tratamento de erros
- Validação de parâmetros

### Total de Testes: 25 testes unitários
- ✅ Todos os testes passando

## Como Executar

### Instalar dependências
```
npm install
```

### Rodar testes
```
npm test
```

### Rodar testes com coverage
```
npm run test:coverage
```

### Rodar testes em modo watch
```
npm run test:watch
```
